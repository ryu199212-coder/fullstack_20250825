- body : 사용자에게 보이는 내용
- 다소문자 / 시작태그와~ 끝나는태그

```bash

<html lang="ko">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>web basic001</title>
</head>
<body>
   안녕 Html 
</body>
</html>
```
